⚡ One-Time Setup Required
Run this SQL in your Supabase dashboard once to create the sync table:
create table if not exists kopit_data ( id text primary key, data text, updated_at timestamptz default now() ); alter table kopit_data enable row level security; create policy "allow all" on kopit_data for all using (true) with check (true);
Go to supabase.com → your project → SQL Editor → paste → Run
